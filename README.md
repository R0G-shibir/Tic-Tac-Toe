# Tic Tac Toe (Java, Swing)

A simple Tic Tac Toe game built with Java and Swing.

**This is a learning project.**

## How to Run

1. **Compile the code:**
   ```sh
   ./build.zsh
   ```
   or manually:
   ```sh
   javac -d bin src/*.java
   ```

2. **Run the game:**
   ```sh
   java -cp bin App
   ```

## Project Structure

- `src/` - Java source files
- `bin/` - Compiled classes
- `build.zsh` - Build and run script

Enjoy playing!
